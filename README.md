# 🐧 Ambiente Linux e Automação (Bash)

Este repositório contém minhas configurações de ambiente (Dotfiles) e exercícios de automação em Shell Script.

### 📂 Organização:
* **01-Fundamentos**: Scripts iniciais e lógica de boot.
* **02-Logica-e-Scripts**: Exercícios de automação e manipulação de variáveis.
* **03-Processamento-de-Texto-Regex**: Uso intensivo de Grep, Sed e expressões regulares para dados biológicos.
* **04-Ferramentas-e-Processos**: Gerenciamento de processos Linux e produtividade com o editor Vim.
* **DOTFILES**: Minhas configurações personalizadas de terminal e editores.

---
*Dominar o terminal é o primeiro passo para pipelines de bioinformática eficientes.*
